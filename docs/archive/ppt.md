# 1. 项目概述

## 1.1 项目定位

**AgentMem** 是一个基于 Rust 开发的企业级智能记忆管理平台，专为 AI 代理和大语言模型应用设计。

### 核心使命
> **让 AI 拥有类人的记忆能力，让智能应用更加智能**

### 解决的核心问题

```
❌ 传统 LLM 的局限性:
├─ 无状态交互     - 每次对话都是全新开始
├─ 上下文窗口限制 - 无法处理长期对话历史
├─ 遗忘用户信息   - 不记得之前说过的偏好和事实
├─ 缺乏个性化     - 无法根据用户历史提供定制服务
└─ 知识更新困难   - 静态知识库，无法动态学习

✅ AgentMem 的解决方案:
├─ 持久化记忆     - 跨会话保存用户信息和偏好
├─ 智能检索       - 毫秒级语义搜索，找到最相关记忆
├─ 无限容量       - 支持千万级记忆，不受窗口限制
├─ 自动学习       - AI 驱动的事实提取和知识更新
└─ 个性化服务     - 基于历史记忆的定制化回答
```

### 2. Episodic-First 检索策略

```
基于认知理论的检索优先级:

Priority 1: Episodic Memory (User scope) - 主要来源 (90%)
  ├─ 权重: 1.2 (提升主要来源)
  ├─ 理论: Long-term Memory
  └─ 作用: 历史对话记忆、用户偏好

Priority 2: Working Memory (Session scope) - 补充上下文 (10%)
  ├─ 权重: 1.0 (正常，因为新鲜)
  ├─ 理论: Working Memory
  └─ 作用: 当前会话上下文

Priority 3: Semantic Memory (Agent scope) - 备选
  ├─ 权重: 0.9 (降低，因为范围更广)
  ├─ 理论: Domain Knowledge
  └─ 作用: 通用知识、领域知识
```

**效果**: 检索准确率提升 26% (LOCOMO 基准测试)

优势:
